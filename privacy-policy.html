隱私權政策

最後更新：2026-07-01

「美食遊覽」（以下簡稱「本服務」）由 Sunpe 維護。本政策說明本服務如何蒐集、處理及利用您的個人資料。依據中華民國《個人資料保護法》（以下簡稱「個資法」）規定，您在使用本服務時享有完整資料當事人權利。

1. 蒐集的資料類型

1.1 您主動提供的資料

  • 使用者推薦店家：店家名稱、地址、類型、營業時間、推薦說明 — 透過首頁右上角「📝 推薦新店家」功能提交

1.2 自動蒐集的資料

  • 本機儲存（localStorage）：
    • 收藏清單（店家名稱陣列）
    • 主題偏好（dark / light / auto）
  • 第三方服務：當您瀏覽本服務時，Supabase Cloud 可能記錄 IP 位址、User Agent 等連線資訊（用於 RLS 安全政策執行）

2. 資料利用目的

  • 店家推薦審核：管理員會檢視您提交的推薦，決定是否收錄至公開地圖
  • 個人化體驗：儲存您的收藏清單與主題偏好，下次造訪時自動載入
  • 服務安全：Supabase Row Level Security (RLS) 政策用於防止未授權存取

3. 資料保存期間

  • localStorage 資料：儲存在您的瀏覽器中，直到您手動清除或反安裝應用
  • 使用者推薦資料：審核通過後保留於公開資料庫；未通過者於 90 日後刪除

4. 第三方服務

本服務使用以下第三方服務，請一併參考其隱私權政策：

  • Supabase Cloud（資料庫 + 認證）：supabase.com/privacy
  • Vercel（靜態託管 + CDN）：vercel.com/legal/privacy-policy
  • Google Maps Place Photo（店家封面圖 CDN）：policies.google.com/privacy

5. 您的權利（個資法第 3 條）

  • 查詢或請求閱覽：您可隨時查看 localStorage 中的收藏與主題資料（透過瀏覽器 DevTools）
  • 請求製給複本：如需推薦資料的副本，請聯絡下方 email
  • 請求補充或更正：推薦資料若需修改，請聯絡管理員
  • 請求停止蒐集、處理或利用：清除瀏覽器 localStorage 即可停止本機資料蒐集
  • 請求刪除：聯絡管理員可要求刪除您的推薦資料

6. Cookie 使用

本服務不使用任何追蹤性 Cookie。所有偏好設定（收藏、主題）皆儲存於 localStorage，不會跨網站追蹤您的行為，亦不投放廣告。

7. 兒童保護

本服務不針對 13 歲以下兒童設計，亦不刻意蒐集其個人資料。如發現兒童資料被誤蒐集，請聯絡管理員立即刪除。

8. 政策變更

本政策更新時將於首頁公告，並標示「最後更新」日期。重大變更將另以醒目方式通知。

9. 聯絡方式
