使用條款

最後更新：2026-07-01

歡迎使用「美食遊覽」（以下簡稱「本服務」）。使用本服務前，請詳閱以下條款。當您開始使用本服務時，即視為同意接受以下所有條款之約束。

1. 服務說明

本服務為非營利性質的宵夜店家資訊整合平台，提供以下功能：

  • 瀏覽高雄、台南、彰化三縣市宵夜店家資訊（164 家）
  • 依縣市、類型、時段篩選店家
  • 收藏喜愛店家（儲存於本機 localStorage）
  • 使用者推薦新店家（由管理員審核後收錄）

本服務不提供線上訂餐、外送、付款、預約座位等交易功能，所有店家資訊僅供參考。

2. 資訊準確性

本服務所列店家資訊（地址、營業時間、照片、評分）係蒐集自 Google Maps 等公開來源，並經人工視覺驗證，惟：

  • 店家可能歇業、搬遷或更改營業時間，實際情況以店家現場公告為準
  • 照片版權屬原上傳者，本服務僅作為資訊整合用途
  • 如有資訊錯誤，請透過 GitHub Issues 回報，管理員將儘速修正

3. 使用者行為規範

使用本服務時，您同意不會：

  • 提交虛假、誤導、誹謗或違法的店家推薦
  • 嘗試未授權存取後端資料庫或其他使用者的資料
  • 對本服務進行反向工程、自動化爬取或干擾正常運作
  • 將本服務用於任何商業用途（未經授權）

4. 智慧財產

本服務之程式碼採 MIT 授權（詳見 GitHub repo）。

店家照片版權屬原上傳者（多為 Google Maps 用戶上傳），僅作為資訊整合引用，未經授權不得轉用。

5. 免責聲明

本服務按「現狀」（AS IS）提供，不對以下事項承擔責任：

  • 店家資訊錯誤導致的任何損失
  • 到店消費產生的任何糾紛
  • 服務暫停、中斷或資料遺失
  • 第三方服務（Supabase、Vercel、Google Maps）中斷造成的影響

6. 服務變更與終止

維護者保留隨時修改、暫停或終止本服務的權利，恕不另行通知。重大變更將於 GitHub repo 公告。

7. 準據法與管轄

本條款之解釋與適用，以中華民國法律為準據法。如有爭議，以台灣台北地方法院為第一審管轄法院。

8. 條款變更

本條款更新時將於首頁公告，並標示「最後更新」日期。繼續使用本服務視為同意變更後之條款。
