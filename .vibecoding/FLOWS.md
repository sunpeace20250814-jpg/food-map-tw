## 3. 關鍵互動

### 互動 1: 縣市切換

- **觸發**: 點城市 chip (高雄/台南/彰化)
- **即時反饋**: 店家卡片過濾顯示, sub-text 更新 (e.g. "高雄 46 家精選"), quick-bar 計數重算
- **結果**: 只顯示該縣市店家
- **禁用條件**: 無
- **失敗時**: 顯示全部 164 家
- **關聯數據**: SHOP_DATA_INITIAL[i].city

### 互動 2: 篩選 chip

- **觸發**: 點 filter chip (全部/24hr/未打烊中/火鍋/日式/小吃/有冷氣)
- **即時反饋**: 店家卡片隱藏不符合的, 計數更新
- **結果**: 顯示符合條件的店家
- **禁用條件**: 該 chip 計數為 0 (灰顯)
- **失敗時**: 顯示空狀態 ("沒有符合的店家")
- **關聯數據**: SHOP_DATA_INITIAL[i].mcat, env, time_24h, late, 22start

### 互動 3: 主題切換

- **觸發**: 點 header 🌙 按鈕
- **即時反饋**: 整頁顏色切換 (dark/light/auto)
- **結果**: 主題切換 + localStorage 記憶
- **禁用條件**: 無
- **失敗時**: 無
- **關聯數據**: localStorage.food-map-tw-theme

### 互動 4: 收藏

- **觸發**: 點 ❤️ 按鈕
- **即時反饋**: ❤️ 變實心 + toast "已加入收藏 ❤️" / "已從收藏移除"
- **結果**: localStorage 更新
- **禁用條件**: 無
- **失敗時**: 顯示 alert (待改 toast)
- **關聯數據**: localStorage.food-map-tw-favorites

### 互動 5: 分享

- **觸發**: 點 📤 按鈕
- **即時反饋**: 嘗試 navigator.clipboard.writeText
  - 成功: toast "已複製到剪貼簿"
  - 失敗: fallback 到隱藏 textarea + execCommand('copy')
  - 全部失敗: toast "分享失敗"
- **結果**: 剪貼簿含店家名 + 站名
- **禁用條件**: 無
- **失敗時**: 顯示錯誤 toast
- **關聯數據**: 卡片 dataset

### 互動 6: 推薦店家

- **觸發**: 點「📝 推薦店家」按鈕
- **即時反饋**: 開 in-page modal (form)
- **結果**: 填表 → 寫入 user_submissions (Supabase)
- **禁用條件**: email 為空
- **失敗時**: inline form 顯示錯誤
- **關聯數據**: Supabase user_submissions table

### 互動 7: 管理員登入

- **觸發**: 點「🔐 管理員」按鈕
- **即時反饋**: 開 in-page modal (email + password input)
- **結果**: supabase.auth.signInWithPassword
- **禁用條件**: email/password 為空
- **失敗時**: modal 內顯示錯誤訊息
- **關聯數據**: Supabase auth.users

### 互動 8: 管理員審核

- **觸發**: 登入後, 點「通過」或「拒絕」按鈕
- **即時反饋**: 開 in-page prompt (審核備註/拒絕原因)
- **結果**: 觸發器自動建 shop (通過) 或 status 改 rejected (拒絕)
- **禁用條件**: 無
- **失敗時**: alert 顯示錯誤 (待改 toast)
- **關聯數據**: Supabase user_submissions.status

### 互動 9: 排序

- **觸發**: 下拉選單 (預設/價位低→高/價位高→低/店名 A-Z)
- **即時反饋**: 卡片重新排序
- **結果**: 卡片順序改變
- **禁用條件**: 無
- **失敗時**: 預設排序

## 6. 邊界和異常

- **輸入為空**: 推薦表單必填欄位 → 紅框 + submit 禁用
- **輸入非法**: email 格式 → 紅框, 不送出
- **輸入過長**: 店家名 max 50, 地址 max 200 → 截斷提示
- **重複點擊**: submit/登入按鈕 disabled 1 秒防重複
- **網絡失敗**: 任何 fetch 失敗 → toast "操作失敗, 請重試"
- **權限不足**: 管理員操作非自己 → toast "您不是管理員"
- **數據為空**: 篩選無結果 → "沒有符合的店家" + 重置篩選提示
- **數據過多**: 164 筆 → 自動分頁 (暫無, 之後加)
- **移動端窄屏**: < 768px → 卡片單欄, header 折疊
